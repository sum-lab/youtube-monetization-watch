# Security Policy

## Public data

This repository is intentionally public and contains only the static website files used for GitHub Pages.
Do not commit API keys, OAuth tokens, private YouTube data, spreadsheets, raw collection scripts, or unpublished evidence.

## Reporting a security issue

For security-sensitive problems, do not open a public issue with secrets or personal information.
Contact the repository owner privately, or open a minimal issue that does not include confidential details.

## User reports

The public issue form is for corrections and channel-status information only.
Do not include passwords, API keys, private revenue screenshots, or personal information.
